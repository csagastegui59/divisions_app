# Divisions-App

## Pre-Requisites

- Yarn version 1.22.5
- Node version 10.19.0
- react version 17.0.1
- antd version 4.12.3
## To run the proyect you need install 

yarn install
### then you run the server with

yarn start
### the routes for aplication are 

http://localhost:3000/

https://divisions-app.netlify.app/
